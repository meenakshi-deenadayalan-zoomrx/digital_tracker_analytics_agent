# Remarkup Template for DTSA Diagnostic Report

Use this template verbatim. Replace `{placeholders}`. Omit sections that don't apply
(e.g. no selector data for web_contents) but keep section headers with "N/A".

```
**DTSA Diagnostic Report**

**Channel:** {channel}
**Platform:** {platform}
**Reported by:** {po_name}
**Investigation timestamp:** {YYYY-MM-DD HH:MM UTC}

---

**== Summary ==**
{One paragraph. Lead with the key finding and confidence level. Include what was ruled out.}

**== Impact Assessment ==**
Priority: **P{N}** (impact score: {0.0–1.0})
Volume deviation: {X}% from baseline
Duration: {e.g. "48 hours, ongoing" or "resolved after deploy at HH:MM"}
Affected scope: {all users | platform X only | specific segment}

**== Evidence Matrix ==**
| Finding | Evidence Sources | Confidence |
|---|---|---|
{One row per finding. Max 6 rows. Format: | description | source1 + source2 | CONFIRMED/LIKELY/INCONCLUSIVE |}

**== Root Cause Analysis ==**
{Detailed explanation with CONFIRMED / LIKELY / INCONCLUSIVE labels on each claim.
Use causal framing: "X changed Y, which caused Z because..."}

**== Volume Data ==**
| Period | Count | vs. Baseline |
|---|---|---|
| Current ({start} – {end}) | {N} | — |
| -7 days ({start} – {end}) | {N} | {+/-X%} |
| -28 days ({start} – {end}) | {N} | {+/-X%} |

{Add platform breakdown table if relevant}

**== Log Analysis ==**
{Error timeline: "First error at TIME, peak at TIME (N/hour), N total over PERIOD. Status: ongoing/resolved."
Or: "No errors found in application or error logs for this period."}

**== Selector Validation ==**
{Table of tested selectors with element_count and bot_detection results.
Or: "Not applicable for {channel} channel."}

**== Relevant Commits ==**
| Commit | Repo | Message | Relevance | Timing vs. Drop |
|---|---|---|---|---|
{List HIGH and MEDIUM commits. Include LOW if nothing else found.}

**== Code Analysis ==**
{Files read, key line numbers, what each read revealed.
Or: "Not required — root cause confirmed from logs/selectors without code analysis."}

**== Recommended Actions ==**
1. {Specific, independently actionable item}
2. {Specific, independently actionable item}
3. {Add as many as needed}

---
//Generated by DTSA v2.0 — {ISO timestamp}//
```

## Remarkup Formatting Notes

- Bold: `**text**` (not `**text**` markdown)
- Italic: `//text//`
- Tables: `| col | col |` with `|---|---|` separator
- Code: ` ``` ` fenced blocks
- Section headers: `**== Section ==**` (not markdown `##`)
- Monospace inline: `##code##`
